* nixos-installer

This will wipe full drive

#+begin_src sh
  go run main.go -device /dev/nvme0n1
#+end_src

Or using flakes

#+begin_src sh
  nix run github:kirillrdy/nixos-installer -- -device /dev/nvme0n1
#+end_src


** Filesystems

By default installer will use ZFS

if you want to use ext4  use -fs ext4 option
